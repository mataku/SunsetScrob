## SunsetScrob

Last.fm client on Android (mainly for my portfolio)

## Features
- Show latest scrobbles
- Show top albums
- Show top artists
- Update NowPlaying (only Apple Music and Spotify)
- Scrobbling (only Apple Music and Spotify)
- Switch themes
- Fake shared element transition
- Autofill
