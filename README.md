## SunsetScrob

Last.fm client on Android (mainly for my portfolio)

## Features
- Show latest scrobbles
- Show top albums
- Show top artists
- Update NowPlaying and Scrobbling (for Apple Music, Spotify, and YouTube Music only)
- Switch themes
- Fake shared element transition
- Autofill
